# Competitive Landscape — AEC Intelligence Hub

*Last updated: March 2026*

## Direct Threats

### Newforma Konekt
- **What:** Project information management + BIM coordination platform
- **Size:** 4.5M users, 1,500+ firms, 20 years in market
- **Strength:** Deep AEC workflow integration, contract change management, RFI/submittal tracking
- **New:** Info Track (launched June 2025) — centralizes conversations, files, tasks, decisions
- **Integrations:** Microsoft 365, SharePoint, Autodesk Construction Cloud, Revit, Outlook
- **Gap:** Enterprise-heavy, document-management DNA, not AI-meeting-intelligence-native. No transcript parsing or proactive AI coaching.
- **Threat level:** HIGH — closest to our vision, but legacy architecture

### Egnyte Project Hub
- **What:** AI-powered project data infrastructure for AEC
- **Size:** 22,000+ customers
- **Strength:** File governance, AI Copilot, native sync with ACC, Procore, Bluebeam, Newforma
- **New:** Project Hub (launched June 2025) — standardized project folder templates, role-based access
- **Gap:** File management company at heart. Not solving meeting intelligence or cross-department communication. No transcript → action item pipeline.
- **Threat level:** MEDIUM-HIGH — great integrations, but wrong starting point

## Adjacent Competitors

### Monograph
- **What:** Fee tracking and project management for architects
- **Size:** 1,800+ firms, 13,000+ users
- **Strength:** Built by architects, MoneyGantt visual budgeting, AI contract parsing
- **Gap:** Financial only. No communication, no meeting intelligence, no cross-platform sync
- **Threat level:** LOW — different problem entirely

### Procore
- **What:** Construction management platform
- **Gap:** Construction-phase focused, heavy, expensive, not architecture-firm specific
- **Threat level:** LOW for our target

### IntoAEC
- **What:** AI-powered PM + CRM for AEC
- **Gap:** Generic PM, not solving the cross-platform hub problem
- **Threat level:** LOW

### ProjectMark (Bolt AI)
- **What:** AEC CRM with AI assistant for proposals/bids
- **Gap:** Business development focused, not project operations
- **Threat level:** LOW

## Niche/Tangential

- **MeltPlan / Melt Code** — AI building code research engine (relevant to our code compliance feature)
- **InspectMind AI** — AI plan review against building codes
- **Mosaic** — AI resource planning for architecture teams
- **BQE Core / Deltek Ajera** — ERP suites for AEC

## Our Differentiation

1. **AI-first, not AI-added:** We start with meeting intelligence, not documents
2. **Zero workflow change:** Watches what teams already do (Teams, email, ACC, Bluebeam)
3. **Architecture-firm specific:** Not generic AEC — built for the 4-department structure
4. **Institutional memory:** "Catch me up" feature — knowledge survives personnel turnover
5. **Proactive, not reactive:** Flags issues before they hit construction phase
