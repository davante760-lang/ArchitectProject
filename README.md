# AEC Hub — Execution Checklist

Mobile-friendly PWA checklist for the AEC Intelligence Hub project.

**Dave × Garth**

## What This Is

A shared execution checklist tracking everything from validation to prototype to go/no-go decision for an AI-powered project coordination hub for architecture firms.

- iOS Add to Home Screen support (standalone web app)
- Touch-optimized with large tap targets
- localStorage persistence — checks survive refresh
- Owner tags (Dave / Garth / Both) and priority levels
- Progress tracking per phase and overall

## Run Locally

```bash
npm install
npm start
```

## Deploy

Deployed on Railway. Root URL serves the checklist.
